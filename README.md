# All-In-One Fitness: Web-Based Scheduling & Registration System

## Overview
The **All-In-One Fitness Web-Based Scheduling & Registration System** is a full-stack web application designed to streamline class scheduling and member registration for a fitness studio with **500+ members**.  
This system replaces manual tracking with a secure, database-driven solution to improve operational efficiency and data accuracy.

---

## Problem
Many fitness studios rely on spreadsheets or manual processes to manage class schedules, registrations, and reporting. These approaches are inefficient, error-prone, and difficult to scale.

This project addresses these challenges by centralizing data management and automating key workflows through a relational database and web interface.

---

## Technologies
- **Database:** SQL, PL/SQL  
- **Backend:** Python  
- **Frontend:** HTML, CSS, JavaScript  

---

## Database Design
- Designed a **7-table relational database** with clearly defined entity relationships
- Enabled accurate data storage, retrieval, and reporting
- Applied normalization to reduce redundancy and improve consistency

---

## Data Security & Integrity
- Implemented **PL/SQL stored procedures** for all database operations
- Restricted direct table access to enhance security
- Ensured reliable and consistent CRUD operations

---

## Features
- Class scheduling and member registration management
- Full **CRUD functionality** (Create, Read, Update, Delete)
- Dynamic dropdown menus for classes and members
- Responsive user interface for faster administrative workflows

---

## Impact
- Streamlined scheduling and registration for **500+ members**
- Reduced manual tracking and administrative overhead
- Improved data accuracy and reporting capabilities
- Enhanced overall user experience for administrators
